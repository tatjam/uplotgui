# uplotgui

A simple "drop in replacement" for uplot that uses GNUplot in the backend, for when the terminal is not enough.

## Installation

## Disclaimer

The project is a weekend project to learn OCaml, parser combinators and make a little useful tool for my personal use, if you find it useful, feel free to use it and report any issues you find! Not all uplot functions are supported, but all arguments are (or should be), so it can be dropped in.
